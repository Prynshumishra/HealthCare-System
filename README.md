# рџ“Њ Project Overview

Prescripto is a modern healthcare platform designed to simplify the interaction between **patients, doctors, and administrators**.  

Patients can browse doctors and book appointments, doctors can manage their schedules and profiles, and administrators can monitor the entire system.

The platform also includes a **Career Portal** that allows users to apply for job opportunities in the healthcare system.
